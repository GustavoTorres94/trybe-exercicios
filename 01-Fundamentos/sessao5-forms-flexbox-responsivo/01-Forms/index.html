5) id = '' - document

7) textarea - descrição/bio/feedback ... cols= colunas rows= linhas
8) checkbox : name, id, valeu - checked signifca valor já marcado;
9) radio: checkbox que perimite so uma opção / o name precisa ser sempre igual, em caso de diferença sera permitido marcar mais de um;
